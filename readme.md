# PDF转图片工具

这是一个简单易用的PDF转图片工具，可以将PDF文件转换为图片格式。

## 主要功能

1. PDF文件预览
   - 支持打开PDF文件并预览内容
   - 支持页面缩放和翻页
   
2. PDF转图片
   - 支持单个PDF文件转换
   - 支持批量PDF文件转换
   - 支持选择输出图片格式（PNG/JPG）
   - 转换后的文件自动保存在原PDF所在目录
   - 生成的图片文件名与源PDF文件名相同

## 使用说明

1. 启动应用后，可以：
   - 点击"打开文件"按钮选择单个PDF文件
   - 点击"批量添加"按钮选择多个PDF文件
   
2. 预览功能：
   - 选择单个PDF文件后可以在预览区查看内容
   - 使用工具栏的缩放按钮调整预览大小
   - 使用翻页按钮或方向键浏览不同页面

3. 转换功能：
   - 选择需要转换的文件
   - 选择输出格式（PNG/JPG）
   - 点击"开始转换"按钮
   - 转换完成后会显示成功提示

## 技术实现

- 使用Python 3.8+
- GUI框架：PyQt6
- PDF处理：pdf2image
- 依赖项：
  - PyQt6
  - pdf2image
  - poppler-utils (需要单独安装)
